# Smart Water Management IoT

This repository contains Arduino and Wokwi-based IoT projects focused on sensing, automation, and real-time feedback systems.

## Projects

### 1. Smart Proximity Warning System
An Arduino-based proximity detection project using sensors to provide warning feedback based on object distance.

### 2. Motion Tracking Servo System
A motion-tracking system using MPU6050, servos, and OLED feedback to detect tilt and control servo movement in real time.

### 3. Smart Water Level Controller
A water tank monitoring and management system using an ultrasonic sensor, LCD, LEDs, buzzer, and relay-based pump control.

## Technologies Used
- Arduino
- Wokwi Simulator
- Ultrasonic sensors
- MPU6050
- Servo motors
- OLED / LCD displays
- Embedded system logic
